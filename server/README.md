Server API

## Technologies

- Node.js
- Express.js
- TypeScript

## API endpoints:

### Companies
- **GET** - localhost:8000/api/companies (_get companies_)

## Run app

1. Install dependencies
``` bash
npm install
```

2. Create ``.env`` file

```js
{
  PORT=8080
}
```

3. Run
``` bash
npm run dev
```
